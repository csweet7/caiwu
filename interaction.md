# 个人投资财务统计应用 - 交互设计

## 核心交互功能

### 1. 投资组合管理器
- **资产添加界面**: 用户可以添加新的投资资产，选择资产类型（A股/美股/虚拟货币）
- **实时价格显示**: 表格显示当前价格，支持手动刷新和自动更新
- **持有份额管理**: 用户可以编辑持有份额，系统自动计算总资产价值
- **多币种显示**: 美股和虚拟币显示美元价格，A股显示人民币价格

### 2. 资产变化记录系统
- **交易记录添加**: 用户可以记录买入/卖出操作，包括日期、数量、价格
- **记录类型选择**: 买入、卖出、分红、转账等不同类型的记录
- **批量操作**: 支持批量添加多个交易记录
- **记录筛选**: 按时间、资产类型、操作类型筛选记录

### 3. 财务统计分析
- **实时资产总览**: 显示当前总资产（美元+人民币分别计算）
- **盈利分析**: 月盈利、年盈利统计，支持不同时间段查看
- **资产分布图**: 饼图显示不同资产类型的占比
- **收益趋势图**: 折线图显示资产变化趋势

### 4. 汇率管理
- **实时汇率**: 自动获取美元兑人民币汇率
- **多币种汇总**: 将人民币资产按当前汇率转换为美元，计算总资产
- **汇率历史**: 显示汇率变化趋势

## 用户操作流程

1. **首次使用**: 用户添加投资资产，设置初始持有份额
2. **日常管理**: 查看实时价格，更新持有份额，记录交易操作
3. **财务分析**: 查看资产分布、盈利统计、收益趋势
4. **数据维护**: 定期更新价格，清理过期记录

## 交互特性
- **响应式设计**: 适配桌面和移动设备
- **实时更新**: 价格自动刷新，无需手动操作
- **数据持久化**: 使用localStorage保存用户数据
- **导出功能**: 支持导出投资报告和交易记录
