# 个人投资财务统计系统

一个功能完整的个人投资管理Web应用，支持A股、美股、虚拟货币的统一管理和分析。

## 🌟 主要功能

### 📊 投资组合管理
- **多资产类型支持**: A股、美股、虚拟货币统一管理
- **实时价格更新**: 每30秒自动更新资产价格（模拟实时数据）
- **多币种显示**: 美股/虚拟币显示美元价格，A股显示人民币价格
- **汇率自动换算**: 实时汇率转换，支持美元和人民币总资产统计
- **盈亏分析**: 实时计算每个资产的盈亏情况和收益率

### 📈 数据分析
- **资产分布图表**: 饼图显示不同类型资产的占比
- **收益趋势图**: 折线图展示投资收益变化趋势
- **关键指标**: 总盈利、收益率、月度/年度盈利统计
- **表现分析**: 最佳/最差表现资产排行
- **详细分析**: 资产权重、成本、市值等详细数据

### 📝 交易记录管理
- **完整记录**: 支持买入、卖出等交易类型记录
- **智能筛选**: 按类型、资产、时间范围筛选记录
- **数据统计**: 交易次数、平均交易额等统计信息
- **数据导出**: 支持CSV格式导出交易记录
- **批量管理**: 支持删除和批量操作

## 🚀 使用方法

### 启动应用
1. 下载所有文件到本地目录
2. 使用Python启动本地服务器：
   ```bash
   python -m http.server 8000
   ```
3. 在浏览器中访问 `http://localhost:8000`

### 基本操作

#### 添加资产
1. 在左侧表单中选择资产类型
2. 选择具体的资产代码
3. 输入持有份额和平均成本
4. 点击"添加资产"按钮

#### 管理投资组合
- 查看实时价格和盈亏情况
- 编辑持有份额或删除资产
- 刷新价格获取最新数据
- 导出/导入投资组合数据

#### 分析投资表现
- 查看资产分布和收益趋势
- 分析最佳和最差表现资产
- 查看详细的资产分析数据

#### 记录交易
- 添加买入或卖出记录
- 按类型、资产、时间筛选记录
- 导出交易记录为CSV文件

## 🔧 自定义配置

### 添加新的资产类型
在 `main.js` 文件的 `commonAssets` 对象中添加新的资产类型和代码：

```javascript
const commonAssets = {
    '美股': [
        // ... 现有资产
        { symbol: 'NEW', name: '新资产' }
    ],
    // ... 其他类型
}
```

### 修改价格更新频率
在 `main.js` 文件中修改 `startPriceUpdates` 函数的定时器：

```javascript
// 每30秒更新一次价格
setInterval(updatePrices, 30000); // 修改为需要的频率
```
